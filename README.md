# Data Visualization and Storytelling

## Objective
Analyze Superstore sales data and create an interactive dashboard.

## Tools Used
- Power BI
- Superstore Dataset

## Dashboard Highlights
- Total Sales Analysis
- Profit Analysis
- Regional Performance
- Customer Analysis
- Product Category Insights

## Key Findings
- Technology category generated highest sales.
- Certain Furniture products produced losses.
- West region performed best.
- Top customers contributed major revenue.
